## 🌸 Featured Projects

### 🩺 VagalSense
> Closed-loop HRV-based panic attack detection with haptic biofeedback

Real-time stress monitoring using wearable sensors with 4-2-6 breathing intervention. Trained on WESAD and DEAP public datasets.

`ESP32` `AD8232` `DRV2605L` `HRV` `Python` `WESAD` `DEAP`

---

### 🧠 NeuroLens
> EEG seizure detection web application with LSTM fine-tuning pipeline

Deployed at RSU research server — Next.js frontend + Python backend with real-time inference.

`LSTM` `Next.js` `Python` `EEG` `Seizure Detection` `PyTorch`

---

### 🔐 FingerVeinNet
> Finger vein biometric recognition system

ResNet18 + ArcFace loss with Frangi filter preprocessing. Built on a 100-volunteer dataset with SQLite gallery and Tkinter GUI.

`ResNet18` `ArcFace` `OpenCV` `Frangi Filter` `SQLite` `Tkinter` `Biometrics`
